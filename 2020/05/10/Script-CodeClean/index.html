Script-CodeClean

磁盘代码清理脚本工具

说明

  • 删除代码目录下的build文件夹以及out开头的文件夹，对于磁盘代码清理特别有效。

  • 由于一些特殊项目原因，需要排除package，cmd目录，以免误删，具体源码可以自定义修改

  • 本脚本只使用linux，window环境可用git-bash环境或者ubuntu-bash运行

使用

shell

参数 codeDir 待清理的目录路径，绝对路径

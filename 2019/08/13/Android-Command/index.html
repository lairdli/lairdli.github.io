    • Android开发-常用Linux命令行工具梳理

    众所周知 Android系统是基于linux内核的，因而linux上的一些命令行基本上Android上也能适用，有时候有些有些阉割的命令也可以通过busybox这个命令工具做补充。本文总结Android开发常用的一些的命令行操作，带你走进不一样的空间体验。命令太多，建议收藏，有需要时搜索就行。

    Adb常用命令

    adb connect ip:port

    连接指定设备

    shell

    adb disconnect

    断开指定设备

    shell

    adb devices

    列出已连接设备

    shell

    adb kill-server

    adb 关闭服务

    shell

    adb start-server

    adb 启动服务

    shell

    adb install

    安装应用 adb install -r -t apk路径

    -r,-t,-d 选填
    -r表示替换掉原来的apk；如果没有-r，否则会提示apk已经存在
    -t表示可安装测试debug版本的apk,否则会提示TEST_ONLY
    -d表示可以降级安装，否则会提示apk版本太低

    shell

    adb uninstall

    卸载应用，后面指定应用包名

    shell

    adb logcat

    抓取系统日志，建议带上时间戳，并写入文件

    shell

    如需结束，按下^C
    shell

    adb root

    获取root权限

    shell

    adb remount

    挂载系统

    shell

    adb reboot

    重启系统

    shell

    adb shell

    进入命令行模式

    shell

    adb -s …

    如果有adb连接有多台设备，可用-s指定设备识别号后，再后面再加命令
    识别号可通过adb devices查看

    shell

    Code

    Android-Linux 常用命令

    系统信息

    busybox uname -m

    显示机器的处理器架构(2)

    shell

    busybox uname -r

    显示正在使用的内核版本

    shell

    cat /proc/cpuinfo

    显示CPU info的信息

    shell

    cat /proc/meminfo

    查看内存使用

    shell

    cat /proc/version

    显示内核的版本

    shell

    cat /proc/net/dev

    显示网络适配器及统计

    shell

    cat /proc/mounts

    显示已加载的文件系统

    shell

    busybox cal 2019

    显示2019年的日历表

    shell

    date

    显示系统日期

    shell

    date -s “yyyymmdd.hhmmss”

    设置日期和时间

    shell

    getprop

    查看系统属性

    shell

    也可以过滤字段 getprop | grep 过滤字符
    shell

    也也可指定属性 getprop 属性名
    shell

    setprop

    设置系统属性,注意有些系统属性设置后重启可能会失效

    shell

    dumpsys

    dump系统所有的信息，由于信息量太大，建议会写入文件

    shell

    dumpsys package

    dump应用信息，一般可以指定包名，查看应用信息

    shell

    dumpsys meminfo

    查看内存信息，可指定应用包名或者进程pid

    shell

    ps

    查看当前系统运行应用进程，可配合grep使用

    shell

    Code

    logcat

    查看日志，建议加上时间戳 ，TAG等

    shell

    &

    一个神奇的指令，命令行+& ，能将命令置于后台，不影响前台输出

    shell

    busybox killall system-server

    重启服务，可模拟快速重启，硬件不重启

    Code

    文件路径

    cd -

    切换到上一次路径

    shell

    ll -a

    列出目录文件，含权限信息,-a 包含隐藏文件

    shell

    mkdir dir

    创建文件夹

    shell

    rm -rf dir

    删除文件夹

    shell

    ln file link

    创建硬链接，拷贝+同步
    硬链接不能指向自身文件系统外的文件，即不能跨硬盘分区(软链接可以)。
    硬链接不能指向目录。

    ln -s file link

    创建软链接，软链接可以类比Windows上的快捷方式

    df

    显示系统挂载盘

    shell

    du -sh *

    显示当前文件夹下子项大小

    shell

    du -sh ./

    显示当前文件夹总大小

    shell

    touch

    touch命令用于修改文件或者目录的时间属性，包括存取时间和更改时间。
    若文件不存在，系统会建立一个新的文件。
    ls -l 可以显示档案的时间记录。

    shell

    chmod 777 file/dir

    修改文件/文件夹读写权限

    shell

    tar

    • tar -cvf archive.tar file1
      压缩

      shell

    • tar -xvf archive.tar
      解压

      shell

    busybox find

    • busybox find -name tmp
      从当前目录搜索文件和目录,指定名称

      shell

    • busybox find -name *prop
      从当前目录搜索后缀为.prop的文件

      shell

    • busybox find -name ‘*.prop’ | grep ‘build’
      从当前搜索文件名含字符build

      shell

    • busybox find -name ‘*.prop’ | busybox xargs grep ‘version’
      从当前搜索文件名为.prop后缀，且文件内容包含version的文件

      shell

      busybox which su

      显示一个二进制文件或可执行文件的完整路径

      shell

      pwd 显示当前文件夹路径全路

      shell

    cat file1

    从第一个字节开始正向查看文件的内容

    shell

    busybox tac file1

    从最后一行开始反向查看一个文件的内容

    shell

    busybox more file1

    查看一个长文件的内容

    shell

    busybox head -2 file1

    查看一个文件的前两行

    shell

    busybox tail -2 file1

    查看一个文件的最后两行

    shell

    busybox tail -f /data/anr/traces.txt

    实时查看被添加到一个文件中的内容

    shell

    echo aaaaa > file

    写入文件内容，覆盖

    shell

    echo bbbb > file

    写入文件内容，追加

    shell

    busybox vi files

    修改文件内容

    shell

    网络信息

    busybox ifconfig

    shell

    ping

    shell

    busybox traceroute ip

    shell

    tcpdump -i eht0 -w file.cap

    监听网卡的网络包，并保存到文件，之后可以用wireshark等软件分析

    shell

    busybox tftp

    文件传输
    mac建议安装TFTP Server
    windows建议安装 Tftp32
    上传文件 busybox tftp -pl remoteip

    shell

    下载文件
    shell

    busybox wget

    下载网络文件

    shell

    busybox md5sum

    查看文件md5值

    shell

    Android系统签名证书的目录是“build/target/product/security”

    shell
